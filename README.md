🛒 Soul Store – E-Commerce Website

This is a simple E-Commerce Website developed using Core PHP and MySQL during my PM-USHA Internship at Dibrugarh University.

🚀 Features
User and Admin login system
Product and category management
Shopping cart system
Order management
Responsive design using Bootstrap
🛠️ Tech Stack
HTML, CSS, JavaScript
PHP
MySQL
Bootstrap

📁 Project Structure
admin/           → Admin panel files  
assets/          → CSS, JS, images  
config/          → Database connection  
includes/        → Common files (header etc.)  
uploads/         → Product images  
user/            → User authentication  

index.php        → Home page  
product.php      → Product details  
cart.php         → Add to cart  
cart_view.php    → View cart  
checkout.php     → Checkout page  
place_order.php  → Order processing  
remove_cart.php  → Remove items  
create_admin.php → Admin setup  
ecommerce_db.sql → Database file
